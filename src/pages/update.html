Regex Search 1.7

Updates:
  • Most of the code base has been converted to TypeScript with the help of ComFreek
  • Fixed a bug where search doesn't work when changing to a different page.
  • Added an options page where you can set whether you want case insensitive to be on or off by default.
Notes:
  • You can open the extension by typing Alt+Shift+F. You can go to the next and previous search results by typing Alt+Shift+N and Alt+Shift+P, respectively. These shortcuts can be configured by going to the bottom of the extensions page and clicking "Keyboard Shortcuts"
  • This extension uses the JavaScript regex engine. I create a regular expression using 'new RegExp(regexp, flags)', where flags are currently either 'g' or 'gi'.
  • This extension is open source and the code can be found here. I'd appreciate any help, whether it's adding new features, fixing bugs, or making the extension look better (including this update page).
  • This update probably introduced a couple of bugs. If you find any, let me know here.
  • If you like this extension, please review it in the Chrome store
